# Netology_hw_encoding

This script prints top 10 words for files in different encoding.

# Quickstart

Example of script launch on Linux, Python 3.5:

```#!bash

$ pip install -r requirements.txt

```

```#!bash

$ python top_words.py

```

Output example:

